Detailed Description

Iterator used to iterate through some or all the options in a CoAP message. The iterator's filter field is used to decide whether all options will be iterated over (filter == GG_COAP_MESSAGE_OPTION_ITERATOR_FILTER_ANY) or only options with a specified number field (filter == <option number>="">).

An iterator has reached the end of its iteration when the option.number field of the iterator is GG_COAP_MESSAGE_OPTION_NONE
