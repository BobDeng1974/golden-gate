uint32_t  report_interval_ms,
GG_DataProbeListener *  listener,
GG_DataProbe **  probe 
)

Create a Data Probe.

Parameters
optionsOr'ed combination of option flags.
buffer_sample_countAmount of samples that are going to get stored. When buffer_sample_count is too small, samples will be squashed. Must be at least 2 to support windowing mode.
window_size_msSliding window size in milliseconds. 0 to disable windowing.
report_interval_msReporting interval in milliseconds.
listenerData probe report listener. NULL for no reports.
probePointer to where the created object will be returned.
Returns
GG_SUCCESS if the object was created, or a negative error code.
