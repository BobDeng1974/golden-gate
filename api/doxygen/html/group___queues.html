Dequeue an item.

This function may be called from any thread. It may block until there is at least one item available in the queue, or a specified timeout expires.

Parameters
selfThe object on which this method is called.
itemPointer to a variable in which the dequeued item will be returned, if any.
timeoutThe maximum time the caller is willing to wait, in milliseconds.
Returns
GG_SUCCESS if an item was dequeued, GG_ERROR_TIMEOUT if the specified timeout expired, or an error code.
