Parameters
selfThe object on which this method is invoked

num_eventsNumber of events in the array
nodeListener Node to store data. If NULL, event dispatcher will allocate and manage memory of the node NOTE: Event Dispatcher will NOT copy over the events. User must ensure events array remains valid A listener can only be registered once. If a listener tries to register again dispatcher will return GG_ERROR_INVALID_PARAMETERS NOTE: Well behaved event listeners should not emit the event they listen to when called back. Listeners that are added during an onEvent callback will be notified only for subsequent events
