GG_Result(* GG_FrameAssemblerInterface::Feed) (GG_FrameAssembler *self, size_t *data_size, GG_Buffer **frame)

Notify the assembler that data has been copied into the feed buffer, and possibly produce a re-assembled frame. If a frame is produced, it must eventually be released by the caller when no longer needed.

Parameters
selfThe object on which this method is called.
[in,out]data_sizePointer to a variable containing, when this method is called, the amount of data fed, and, when the method returns, the number of bytes consumed by the frame assembler (always <= the amount passed in).
[out]framePointer to the address of a variable where an assembled frames may be returned, if a complete frame was found.
Returns
GG_SUCCESS if the method succeeded, or a negative error code.

void(* GG_FrameAssemblerInterface::GetFeedBuffer) (GG_FrameAssembler *self, uint8_t **buffer, size_t *buffer_size)

Get the address and size of the buffer into which data can be fed. After this function returns, the caller may write up to *buffer_size bytes into the returned buffer before calling GG_FrameAssembler_Feed
