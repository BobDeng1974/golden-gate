size_t  input_size,

bool  url_safe 
)

Decode a base64 string into a byte array.

Parameters
inputBase64 string to decode.
input_sizeLength of the input string (if the string is null-terminated, passing 0 will automatically measure the length of the string).
outputOutput buffer.
output_sizePointer to a variable containing on input the size if the output buffer and on output the number of bytes written. If the buffer size is too small, the buffer size needed is returned instead.
url_safeBoolean indicating whether or not to use the URL-safe character set instead of the standard one.
Returns
GG_SUCCESS if the input could be decoded, GG_ERROR_NOT_ENOUGH_SPACE if the output buffer was too small, or GG_ERROR_INVALID_FORMAT if the input is not valid base64.
