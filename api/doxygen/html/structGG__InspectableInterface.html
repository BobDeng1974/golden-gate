GG_Result(* GG_InspectableInterface::Inspect) (GG_Inspectable *self, GG_Inspector *inspector, const GG_InspectionOptions *options)

Inspect an object. The inspected object will call back an inspector for each field that may be inspected. The inspector may be called back 0 or more times, depending on the amount of detail the object has. When this call returns, no futher callback will be invoked on the inspector.

Parameters
selfThe object on which this method is invoked.
inspectorThe inspector that should be called back during the inspection.
options.Options for the inspection. May be NULL when no specific option is needed.
Returns
GG_SUCCESS if the call succeeded, or a negative error code.
